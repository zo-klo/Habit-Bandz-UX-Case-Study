# Habit Bandz UX Case Study

This repository contains a public-safe portfolio version of a UX and product insight project for Habit Bandz.

## Contents

- `CASE_STUDY.md`: main case study narrative
- `public_notebooks/01_habitbandz_user_insight_workflow.ipynb`: cleaned research and insight notebook using synthetic data
- `public_notebooks/02_persona_simulation_exploration.ipynb`: exploratory persona simulation notebook
- `public_notebooks/habitbandz_portfolio_utils.py`: helper functions for the notebook workflow
- `public_notebooks/data/habitbandz_synthetic_feedback.csv`: synthetic sample dataset

## Notes

- This repository uses synthetic data and redacted configuration so the workflow can be shared publicly.
- The simulation notebook is intended for exploratory scenario testing and is not a substitute for user research.
- Any API key should be supplied through an environment variable rather than committed to the repository.
