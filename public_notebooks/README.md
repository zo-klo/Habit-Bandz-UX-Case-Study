# Public Notebooks

This folder contains a GitHub-friendly version of the Habit Bandz workflow using synthetic data and redacted configuration.

Files:

- `01_habitbandz_user_insight_workflow.ipynb`: main portfolio notebook for research synthesis and product insight
- `02_persona_simulation_exploration.ipynb`: separate exploratory notebook for persona-based scenario simulation
- `habitbandz_portfolio_utils.py`: helper functions shared by both notebooks
- `data/habitbandz_synthetic_feedback.csv`: synthetic sample dataset used in the public workflow

Notes:

- The data in this folder is synthetic and intended only for portfolio demonstration.
- Any model key should be provided through an environment variable or replaced where the notebook prompts for `[INSERT KEY HERE]`.
- The simulation notebook is framed as exploratory analysis and not a substitute for user research.
